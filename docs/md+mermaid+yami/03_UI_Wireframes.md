## 3. עקרונות עיצוב

### 3.1 צבעים
- **Primary:** #2563eb (כחול)
- **Secondary:** #10b981 (ירוק)
- **Accent:** #f59e0b (כתום)
- **Error:** #ef4444 (אדום)
- **Background:** #f8fafc
- **Text:** #1e293b

### 3.2 טיפוגרפיה
- **Headers:** Heebo Bold
- **Body:** Heebo Regular
- **Numbers:** Rubik

### 3.3 רספונסיביות
- Mobile-first design
- Breakpoints: 640px, 768px, 1024px

### 3.4 RTL Support
- כל הטקסטים בעברית מיושרים לימין
- Navigation bar משמאל לימין
